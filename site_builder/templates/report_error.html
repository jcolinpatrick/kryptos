Report an Error

Found a mistake in our elimination database? We take accuracy seriously. Every result should be reproducible and correctly described.

How to Report

  1. Navigate to the specific elimination page that contains the error.
  2. Click the "Report it on GitHub" link at the bottom of the page. This opens a pre-filled GitHub Issue with the elimination ID.
  3. Describe the error you found: wrong parameters, incorrect verdict, reproduction command fails, scope limitations missing, etc.
  4. Submit the issue. We will investigate and update the database if confirmed.

What to Report

  • Incorrect results: Reproduction command gives different output than claimed.
  • Missing scope limitations: The elimination claims to rule out something broader than it actually tested.
  • Wrong parameters: Period range, key model, or cipher type is misidentified.
  • Stale information: New public information changes the interpretation of results.
  • Bug in the code: Indexing errors, sign convention mistakes, or other implementation bugs.
